# CorpSync – Corporate Service Dashboard

A high-performance, responsive corporate dashboard built with **React**, **TypeScript**, **Redux Toolkit**, and **Recharts**. Designed to demonstrate front-end engineering skills aligned with enterprise-grade corporate service systems.

## 🚀 Features

- **Overview Page** — Metrics cards, weekly activity bar chart, department pie chart, recent tasks table
- **Employee Directory** — Searchable employee cards with status indicators
- **Task Manager** — Filter by status, update tasks via Redux, priority badges
- **Announcements Board** — Company-wide announcements with category tags
- **Settings Page** — Profile editing with toggle controls
- **Collapsible Sidebar** — Smooth animated navigation
- **Global Search** — Filters employees and tasks in real time

## 🛠 Tech Stack

| Layer | Technology |
|---|---|
| UI Framework | React 18 + TypeScript |
| State Management | Redux Toolkit |
| Charts | Recharts |
| Icons | Lucide React |
| Build Tool | Webpack (via CRA) |

## 📦 Getting Started

```bash
git clone https://github.com/faizanmahfooz07/corporate-dashboard.git
cd corporate-dashboard
npm install
npm start
```
